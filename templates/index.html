Task 1 - Python & Data Science

AI Chatbot Web Application

This high-fidelity application serves as the complete project submission for **Internship Task 1**, demonstrating a robust application built with a modern backend foundation and a highly customized UI.

Key Implemented Features:
  • SQLite Database Persistence: Full local database storage of chat threads, messages, and configurations.
  • Five Vibrant Themes: CSS visual style configurations using modern gradients and glassmorphic designs.
  • Adaptive AI Personas: Distinct personalities (Assistant, Coach, Coder, Novelist) with corresponding simulated context.
  • Speech-to-Text (STT): Microphone recognition using Web Speech APIs.
  • Text-to-Speech (TTS): Clear narration of chatbot outputs.
  • Rich Markdown Parsing: Frontend rendering of markdown structures (bold, lists, titles, code syntaxes).
